# 🎮 PC방 키오스크 시스템

TCP 기반 키오스크, 사용자, 관리자 간 통신 시스템 및 QR 인증 기능을 포함한 PC방 관리 프로그램입니다.

---

## 📌 프로젝트 소개
본 프로젝트는 PC방 환경을 가정하여, 키오스크를 통한 좌석 선택 및 결제, 사용자 PC 이용 관리, 관리자 서버 제어, QR 인증 기능까지 포함한 통합 시스템을 구현한 프로젝트입니다.  
각 구성 요소 간 TCP 통신을 통해 실시간 데이터 처리를 수행하도록 설계했습니다.  
클라이언트-서버 구조를 기반으로 사용자, 관리자, 키오스크 간 데이터 흐름을 구현했습니다.

---

## 🧩 시스템 구성
- Kiosk (키오스크)
- User Client (사용자 PC)
- Admin Server (관리자)
- QR Module (QR 인증)
- TCP 통신 기반 구조

---

## 🔄 시스템 흐름
1. 사용자가 키오스크에서 로그인 및 좌석 선택  
2. 결제 및 이용 시간 등록  
3. 사용자 PC에서 로그인 후 이용 시작  
4. 관리자 서버에서 전체 좌석 및 상태 관리  
5. QR 인증을 통한 사용자 인증 처리  

---

---

## 💡 주요 기능

### 🎮 키오스크
- 로그인 및 회원 관리  
- 좌석 선택  
- 결제 (카카오페이, 네이버페이 등)  
- 이용 시간 충전  

### 💻 사용자 PC
- 로그인 및 이용 상태 관리  
- 관리자와 채팅 기능  
- 메뉴 선택 및 이용 제어  

### 🖥 관리자 서버
- 전체 좌석 상태 관리  
- 사용자 접속 및 이용 상태 확인  
- TCP 서버 기반 통신 처리  

### 📷 QR 인증
- 카메라 기반 QR 코드 인식  
- 사용자 인증 처리
